# 弋阳县/万年县充换电设施规划项目

> **项目性质**：江西省上饶市弋阳县、万年县充换电基础设施规划与申报数据整理工作台
> **工作阶段**：数据整理 → 入库 → 可视化（当前阶段）→ 规划报告编写
> **最后更新**：2026-03-08
> **数据截止**：2025年底（2026年规划数据为申报阶段）

---

## 一、项目背景

弋阳县与万年县（均隶属江西省上饶市）正在推进县域新能源汽车充换电基础设施建设申报工作，
覆盖 2026-2028 年建设规划。本项目对申报材料中的 Excel 数据、Word 政策文档进行
系统整理，建立结构化数据库，并生成可视化仪表盘与地图，为后续规划报告编写提供支撑。

**弋阳县基本情况（2025年）**
- GDP：203.94 亿元
- 常住人口：约 32.94 万人（2024年数据）
- 汽车保有量：5.986 万辆
- 新能源车保有量：0.3225 万辆，渗透率 5.39%

## 八、地图实现说明

### 8.1 技术选型
- **地图服务**：天地图（国家地理信息公共服务平台）
- **API 版本**：天地图 JavaScript API v4.0
- **坐标系统**：GCJ-02（火星坐标系）

### 8.2 聚光灯效果实现
采用 **SVG 叠加层 + Mask 技术**实现县域聚光灯效果：

1. **SVG 叠加层**：在地图容器上方创建全屏 SVG 元素
2. **SVG Mask**：定义遮罩，白色区域显示遮罩，黑色区域透明
3. **县域路径**：使用县域边界坐标绘制 SVG path
4. **动态更新**：监听地图移动/缩放事件，实时更新 SVG 路径坐标
5. **视觉效果**：
   - 县域内：透明（显示原始地图）+ 浅蓝色填充 + 蓝色描边
   - 县域外：黑色半透明遮罩（70% 不透明度）

### 8.3 数据来源
- **现有充电站**：数据库 `stations_urban_coords` 表（16 座，精确坐标）
- **加油站**：数据库 `gas_stations` 表（26 座，需地理编码）
- **规划站点**：数据库 `stations_planned` 表（34 站，需地理编码）
- **县域边界**：天地图行政区划 API（运行时获取，如失败使用默认边界）

### 8.4 已知限制
- 加油站和规划站点坐标为模拟数据（随机分布在县城中心附近）
- 实际部署需要补充精确的地理坐标数据
- 天地图行政区划 API 可能因 WAF 拦截导致边界获取失败

## 九、注意事项

1. **编码**：所有 Python 脚本首行均有 `sys.stdout.reconfigure(encoding='utf-8')`，
   必须用 `python scripts/xxx.py` 运行，不要用 `python3`（Windows 环境）
2. **原始数据**：`~$` 开头的文件是 Office 临时锁定文件，所有脚本已过滤
3. **数据库**：每次运行 `03_import_db.py` 会 `DROP TABLE` 重建，属于全量刷新逻辑
4. **地图 Key**：API Key 为 Web JS 类型，不能用于服务端 REST 调用（会报 `USERKEY_PLAT_NOMATCH`）
5. **PDF 文件**：3 个 PDF 体积较大（含地图图纸），未转换，如需解析建议分页读取
